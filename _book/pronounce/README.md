# 发音

## 符号解析

| 符号 | 说明 | 示例 |
| :--- | :--- | :--- |
| 加粗字体 | 重读 |  |
| ~    | 同化现象 | taught~you |
| -    | 连读 | write-a |
| 删除线 | 省略 | ge~~t~~ married |
| ↑    | 升调 | shall we↑ |

## 连读策略

l + 元音（多 “了”） all of

n + 元音（多 “呢”）an apple

r + 元音（多 “若”）your eyes

t + 元音（变 d）out (d) of

相同辅音（读一个）bus stop

爆破音 + 辅音（憋住气）look good

/i/ + 元音（多 /j/）see (j) us

/u/ + 元音（多 /w/）do (w) it

t + y（变 /tʃ/）meet you

d + y（变 /dʒ/）would you

s + y（变 /ʃ/）miss you
